How It Works

Host AI Agents Advise and MentorExecution AI Agents Execute

You stay in Claude Code or Codex. The host AI agent shapes the task, Sidekick routes execution to Forge or Kay, and the host AI agent reviews, corrects, and mentors the execution AI agent before reporting back.

Advisor-Style Delegation

The host AI stays in the advisor role: it breaks down work, delegates to Forge or Kay, then synthesizes and reviews the result.

Session-Gated Hooks

Forge and Kay enforcement stays dormant until delegation starts for the current session, then activates through the session marker.

Traceable Progress

PostToolUse hooks emit compact Forge or Kay summaries, while `.forge` and `.kay` indexes keep durable task lookup metadata.

Mentoring Loop

Host review turns repeated mistakes, project patterns, and task feedback into standing agent guidance so Forge and Kay improve over time.

Installation

Install Once, Delegate Per Session

Install the Sidekick plugin in your host. The plugin provides the shared skills, hooks, and host wiring; Forge and Kay readiness is checked when delegation starts.

Plugin install
# Claude Code
/plugin install alo-labs/sidekick

# Codex / Kay
codex plugin marketplace add alo-labs/sidekick
# Then enable Sidekick from the Codex plugin surface

Restart your host app after installation so the hook surface is loaded before starting a Forge or Kay session.

Setup

Three Steps, Then Delegate

For the user, setup is API access, plugin install, then per-session delegation. Sidekick checks runtime readiness when delegation starts.

1

Get API Access

Prepare the backend path you want Kay or Forge to use. MiniMax.io covers MiniMax M2.7 access, and OpenCode Go remains available for multi-AI compatibility.

MiniMax.io: https://platform.minimax.io/subscribe/token-plan
OpenCode Go: https://opencode.ai/go
2

Install Sidekick

Install the plugin through the host's plugin surface, then restart the host so Sidekick hooks load before a Forge or Kay session.

Claude: /plugin install alo-labs/sidekick
Codex: codex plugin marketplace add alo-labs/sidekick
3

Delegate Per Session

Start the agent workflow for the current session. The host stays in the advisor and mentoring role while Forge or Kay performs implementation work.

Forge: /forge:delegate
Kay: /kay:delegate
