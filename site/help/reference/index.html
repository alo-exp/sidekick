Reference

Current command, runtime, state, generated bundle, registry, and testing reference for Sidekick 0.7.0.

Delegation Commands

CommandPurpose
/sidekick:kay-delegateActivate Kay delegation for the current host session.
/sidekick:kay-delegate xiaomiActivate Kay with Xiaomi routing for the session.
/sidekick:kay-delegate ocgActivate Kay while forcing OpenCode Go routing.
/sidekick:kay-stopStop Kay delegation and clear the Kay session marker.
/sidekick:codex-delegateActivate Codex delegation through the local OpenAI Codex CLI.
/sidekick:codex-stopStop Codex delegation and clear the Codex session marker.

Runtime Commands

RuntimeCommandNotes
Kaykay execPrimary Kay child execution path. Sidekick installs and repairs the Kay runtime.
Kay loginkay login --provider opencode-go --with-api-keyConfigures Kay provider credentials when needed.
Codexcodex execLocal OpenAI Codex CLI path. Sidekick uses gpt-5.4-mini with extra-high reasoning.
