How Can We Help You?

Sidekick ships Kay and Codex. Claude Code, Codex, and Cursor hosts can activate either sidekick for the current session. Use /sidekick:kay for Kay, /sidekick:codex for Codex, and the matching stop command before switching. If you're not sure which path to read first, start with Compatibility.
