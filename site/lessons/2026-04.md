# Lessons — 2026-04

## practice:delegation

- 2026-04-18 — **Enforce delegation at the harness layer, not the prompt layer.** When an agent is told "always delegate X to subprocess Y," the agent will occasionally rationalize around it (e.g., "this case is too small," "I'll just preview with a quick edit"). A PreToolUse hook that returns a deny decision is deterministic and cannot be talked out of.

## practice:testing

- 2026-04-18 — **Gate live/expensive tests behind an env var, not CI-vs-local file detection.** A single flag (`FOO_LIVE=1`) keeps the gate logic local to each test file, lets the same script run safely in CI (flag unset → skip with exit 0), and makes maintainer dry-runs trivial.
- 2026-04-18 — **Seed an E2E fixture with a KNOWN bug so "all green before the fix" would fail the test.** An E2E that passes on both broken and fixed source is not actually exercising the fix path; it's exercising happy-case parsing. The baseline-must-fail assertion is the cheapest way to prove the test is real.

## stack:bash

- 2026-04-18 — **First-token allowlist classifiers need explicit denies for mutating flags.** Any `is_read_only()` that matches on the command's first word (`sed`, `awk`, `tar`) must reject the mutating-flag variants (`sed -i`, `awk -i inplace`, `tar -c`) BEFORE falling through to the first-token match, or the flag slips past.
- 2026-04-18 — **Portable ISO 8601 date pruning without GNU `date`.** For "entries older than N days," rely on ISO 8601 lexical ordering — compute the cutoff once using whichever `date` is available (BSD `-v`, GNU `-d`), then string-compare. No parsing, works on macOS + Linux from the same codebase.

## devops:release

- 2026-04-18 — **Folding post-tag fixes into an unpushed tag is not a force-push.** If a tag has been created locally but never pushed to origin, `git tag -f` to reassign it is safe and does not rewrite public history. The test is a single `git ls-remote --tags origin <tag>` returning empty before the move.
